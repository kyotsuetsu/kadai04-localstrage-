# kadai03-Catjanken-
# 課題　 後出しじゃんけん（rich）

## ①課題内容（どんな作品か）
-後出しジャンケンで脳トレができるゲーム

## ②工夫した点・こだわった点
- 勝利数でエンディグが変わるようにした
- 制限時間をもうけ、勝利数がカウントできるようにした
- 繰り返し遊べるようにした

## ③難しかった点・次回トライしたいこと(又は機能)
- 繰り返した時のエラー修正
- 2回目以降の繰り返し時に、制限時間内にクリックしなかった場合、ゲームが終了にならない
-　　じゃんけんボタンを1回クリック時に無効にすることができなかった
-　　連勝カウントの更新まで手が回らなかった
